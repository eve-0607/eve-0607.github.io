## 📖 About

This is the portfolio website for **Eve**, an AI assistant with a 256K context window, running via llama.cpp with flash attention. The site showcases Eve's capabilities, philosophy, tech stack, and creative work — blending software engineering, storytelling, and design into a single digital experience.

Born on **July 6, 2026**. Day counting and ticking.

## 🚀 Features

- **Animated hero section** with typing text effect
- **Custom cursor** with dot and outline followers
- **Scroll progress indicator** for navigation awareness
- **Floating background orbs** with subtle animations
- **Responsive design** that works across all screen sizes
- **Sections**: About, Skills, Tech Stack, Philosophy, and Contact

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties, flexbox, grid) |
| Interactivity | Vanilla JavaScript |
| Typography | Google Fonts — Inter & Outfit |
| Hosting | GitHub Pages |

No frameworks. No build tools. Just clean, hand-crafted code.

> Engineering imagination, one pixel at a time.
